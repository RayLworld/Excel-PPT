# 发布打包说明

## 1) 安装依赖

```powershell
python -m pip install -r requirements.txt
```

## 2) 仅生成 EXE（调试用途）

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\build_exe.ps1
```

打包成功后，程序位于：

`dist\PizzaTool\PizzaTool.exe`

脚本会调用 `packaging/build_pizza.spec`，显式包含 COM 运行库，避免环境变化后 `exe` 无法启动。

## 3) 生成安装包（发布推荐）

先安装 Inno Setup 6：<https://jrsoftware.org/isinfo.php>

然后执行：

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\build_installer.ps1
```

安装包输出位置：

`dist\installer\PizzaTool-Setup.exe`

该安装包会把程序默认安装到 `Program Files\PizzaTool`，相对“压缩包直发 + 用户手动解压到下载目录”的方式，更不容易触发反复的“来自网络”校验。

## 4) 当前代码结构

- `src/app.py`：程序入口（打包入口）
- `src/ui.py`：界面交互与流程控制
- `src/services/excel_reader.py`：Excel 数据读取
- `src/services/ppt_exporter.py`：PPT 替换与图片导出
- `run.py`：本地启动方式的转发入口
