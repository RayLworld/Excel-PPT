# Excel-PPT
基于 Python + Tkinter 的可视化批量 PPT 内容替换与导出工具。帮妈妈做的办公小项目，适合拿来批量生产贺报或名牌。
### Features
读取 Excel 数据，批量替换 PPT 模板指定字段
按幻灯片尺寸导出高清 PNG
支持 WPS / Microsoft PowerPoint 静默后台处理
自动依赖安装、进度条与日志、进程安全清理
### Environment
Windows
Python 3.7+
WPS 或 Microsoft Office（PPT 组件）
### Usage
选择 PPT 模板、Excel 数据文件、输出目录
设置表头行并读取 Excel，选择待替换字段
启动批量处理，自动导出图片
### Dependencies
pandas, python-pptx, psutil, pywin32, openpyxl
